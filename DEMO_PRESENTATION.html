Dual-model pipeline: YOLO11-Medium for fast real-time detection + GroundingDINO for zero-shot semantic understanding of arbitrary product descriptions.

Backend API
FastAPI async backend with 12+ REST endpoints and WebSocket support. Auto-generates Swagger docs. CORS enabled for all frontend clients.
FastAPI Uvicorn WebSocket REST

Demand Forecasting
Per-SKU XGBoost models with 730-day M5-style synthetic data. Lag features, rolling statistics, promo flags. Z-score safety stock calculation.
