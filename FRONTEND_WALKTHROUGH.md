# Shelfwise Frontend Initialized

The frontend for **Shelfwise** has been successfully initialized using React and Vite. It features a premium, minimalist design with dark mode, glassmorphism, and smooth animations.

## Key Features
- **Modern UI**: Custom CSS design system with deep navy tones and electric blue accents.
- **Animations**: Powered by `framer-motion` for smooth reveal effects.
- **Icons**: Professional icon set from `lucide-react`.
- **Aesthetic Assets**: Includes a high-end, AI-generated hero image representing smart shelving.

## Project Structure
- `frontend/src/App.jsx`: Main landing page component.
- `frontend/src/index.css`: Design system and global styles.
- `frontend/public/hero.png`: AI-generated premium hero asset.

## How to Run
To start the development server, navigate to the frontend directory and run:
```bash
cd frontend
npm run dev
```

> [!NOTE]
> If you encounter Node.js binding errors, ensure your environment is compatible with Vite 8.0+. You may need to run `npm install` again if dependencies were interrupted.
